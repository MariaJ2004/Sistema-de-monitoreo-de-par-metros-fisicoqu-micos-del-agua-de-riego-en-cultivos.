# Sistema-de-monitoreo-de-par-metros-fisicoqu-micos-del-agua-de-riego-en-cultivos.
 # Descripción del proyecto

Este proyecto consiste en el desarrollo de un sistema embebido IoT para la medición de parámetros físico-químicos del agua de riego en cultivos. El sistema permite monitorear TDS, turbidez y temperatura mediante sensores integrados a microcontroladores Arduino en las primeras fases y una ESP32 en la etapa final, lo que proporciona mayor capacidad de procesamiento y conectividad.

# Funcionalidades principales

Adquisición de datos de TDS, turbidez y temperatura en tiempo real.

Visualización local mediante pantalla LCD.

Almacenamiento en tarjeta MicroSD para registro histórico.

Reloj en tiempo real (RTC) para marcar cada medición con fecha y hora.

Conectividad Wi-Fi mediante ESP32 para el envío automático de datos a la nube a través de ThingSpeak.

Sistema IoT diseñado para operar de forma continua en ambientes agrícolas.

# Interfaz de usuario

Se desarrolló una interfaz gráfica en MATLAB que permite visualizar, analizar y gestionar los datos recopilados. Incluye autenticación mediante usuario y contraseña para asegurar el acceso a la información.

# Contexto del proyecto

El sistema fue implementado como parte de un proceso de investigación desarrollado en colaboración con el Laboratorio de Tratamiento de Aguas de la Facultad de Ingeniería Ambiental de la Universidad Pedagógica y Tecnológica de Colombia (UPTC). Su objetivo principal es apoyar el análisis de calidad del agua en sistemas de riego, ofreciendo una herramienta automatizada, portable y eficiente.

# Sistema-de-monitoreo-de-par-metros-fisicoqu-micos-del-agua-de-riego-en-cultivos.
Sistema IoT para medir TDS, turbidez y temperatura en agua de riego. Desarrollado con Arduino y ESP32 (Wi-Fi). Incluye pantalla LCD, registro en MicroSD, RTC y envío a ThingSpeak. Se integró una interfaz en MATLAB para visualización y análisis. Proyecto apoyado por el laboratorio de aguas UPTC.
